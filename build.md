# Запуск и сборка

## Задание 2-3
Обычный gradle проект, написанный в IntelliJIdea

Можно откыть IDEA и запустить файл `MainCLI.kt`

Если хочется из консоли, в папке с проектом выполнить
```
./gradlew build
./gradlew installDist
./build/install/LL/bin/LL
```

Для запуска тестов
```
./gradlew test
```
